# Ruma

The wallet for humans. Simple, easy crypto self-custody on Ethereum.

## Overview

At Ruma, we identified a common problem when talking crypto to our friends and family. They were LOST.
We aim to become the most used, secure, self-custody crypto wallet on Earth.

We tailored Ruma for humans, making the design super simple, familiar, and easy to use

## Development

1. Clone the repo
2. `cd extension`
3. `pnpm install`
4. `pnpm dev`
